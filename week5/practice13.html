전공 용어 사전(term)

전공관련 용어 3개를 설명하시오.

TCP
서로 다른 시스템을 가진 컴퓨터들을 서로 연결하고, 데이터를 전송하는 데 사용하는 통신 프로토콜들의 집합임.
UDP
인터넷에서 정보를 주고 받을 때, 서로 주고 받는 형식이 아닌 한쪽에서 일방적으로 보내는 방식의 통신 프로토콜임.
XML
인터넷 웹페이지를 만드는 HTML을 획기적으로 개선하여 만든 언어임. 홈페이지 구축기능, 검색기능 등이 향상되었고, 웹 페이지의 추가와 작성이 편리해짐.
